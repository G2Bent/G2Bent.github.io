基于「VitePress」搭建的极简博客 #

效果预览 #

https://jexlau.github.io/blog/

安装 #

bash
git clone https://github.com/JexLau/blog.git
cd blog

# 安装依赖
yarn

# 在本地启动服务器
yarn dev

# 构建静态文件 > .vitepress/dist
yarn build
